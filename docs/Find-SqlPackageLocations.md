# Find-SqlPackageLocations

## SYNOPSIS
Lists all locations of SQLPackage.exe files on the machine

## SYNTAX

```
Find-SqlPackageLocations
```

## DESCRIPTION
Simply finds and lists the location path to every version of SqlPackage.exe on the machine

### EXAMPLE 1
```
Find-SqlPackageLocations
```
